# kj202590 — 固收+ ETF 轮动策略

> 来源：聚宽文章 [固收+](https://www.joinquant.com/post/50559)，作者：开心果

---

## 策略思路

本策略属于**固收增强（固收+）**风格，以债券ETF为底仓，辅以少量权益类和黄金ETF做收益增强，通过**静态固定权重 + 定期再平衡**的方式运行。

### 核心逻辑

1. **固定权重配置**：预先设定各资产目标仓位，不做动态择时。
2. **再平衡触发条件**：每日收盘前检查，当某资产的实际持仓价值与目标价值偏差超过 **15%** 且偏差股数超过 **100手** 时，触发再平衡到目标仓位。
3. **卖出先于买入**：先处理需要减仓的品种（`balance` 值为负），再用释放出的资金补充需要加仓的品种，避免资金不足。

### 资产配置（默认权重）

| ETF 代码 | 品种 | 目标权重 |
|---|---|---|
| 511010.XSHG | 国债ETF（底仓） | 70% |
| 518880.XSHG | 黄金ETF | 14% |
| 510880.XSHG | 红利ETF | 8% |
| 513100.XSHG | 纳指ETF | 8% |

底仓债券占 **70%**，权益+黄金合计 **30%**，整体风险偏好偏保守。

## 关键参数

| 参数 | 值 | 说明 |
|---|---|---|
| `g.weights` | 见上表 | 各品种目标仓位权重，可自行调整 |
| 再平衡阈值 | 15% | 偏差超过目标仓位的 15% 才触发交易，减少无效换手 |
| 最小偏差手数 | 100 份 | 偏差不足 100 份（10万份以下小额）不交易，降低佣金损耗 |
| 运行时间 | 09:35 | 每日开盘后 5 分钟执行，避开集合竞价波动 |
| 滑点 | 0.002 | 双向固定滑点 |
| 佣金 | 万分之二（双向） | ETF 基金佣金，最低 5 元 |
| 基准 | 511010.XSHG | 国债ETF |

## 适合的场景

- **震荡或熊市**：债券底仓提供稳定收益，权益下跌时黄金和债券起到对冲作用。
- **全球资产轮动不确定时期**：纳指+红利+黄金+国债的组合覆盖了多种宏观情景。
- **风险偏好偏低的投资者**：最大回撤相对股票策略更小，适合稳健型账户。
- **不适合单边牛市**：权益仓位仅30%，在强势牛市中涨幅有限，会跑输纯股票策略。

## 运行频次

| 维度 | 说明 |
|---|---|
| 调仓触发 | 每日检查，满足偏差阈值才实际下单 |
| 交易时间 | 每日 09:35，日内执行一次 |
| 实际调仓频率 | 取决于市场波动，低波动时可能数周不调仓 |
| 换手率 | 较低，15% 阈值过滤了大量小幅波动 |

## 移植到 QMT 的注意事项

1. `get_current_data()` 需替换为 QMT 对应的实时行情接口。
2. `order_target_value()` 需替换为 QMT 的目标市值下单方法。
3. `context.portfolio` 相关的持仓、资金字段需映射到 QMT 账户对象。
4. `run_daily` 定时任务需在 QMT 策略框架中用对应的定时器实现。
5. ETF 的 `min_commission=5` 在实盘中请确认券商实际佣金标准。
