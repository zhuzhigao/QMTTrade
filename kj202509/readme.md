# 金阳光 QMT 全天候轮动策略 (All-Weather Strategy) v2.0

## 📝 核心策略逻辑总结
本策略是一个融合了**“宏观动量择时”**、**“微观基本面选股”**与**“动态环境识别（猴市监控）”**的高级混合量化框架。它不仅能在顺风时通过大小盘动量轮动捕捉 A 股核心资产或微盘溢价，还能在逆风（动量皆负 / 均线破位）或市场陷入无序震荡（猴市）时，自动切入防御模式（等权配置外盘黄金与纳指 ETF），并自带日内 8% 硬止损的最后防线。

---

## 📈 策略的细节

策略引擎以 `handlebar` 为核心，以时间节点为驱动，细分为四大独立运作的模块：

### 1. 模块 0：全局猴市巡检 (每日 09:31)
* **动态环境识别**：每日开盘后即刻计算大盘（沪深300）近 20 日的**考夫曼效率系数 (ER)** 与 **变异系数 (CV)**。
* **熔断与挂起**：若发现市场“趋势极弱且波动极大”（判定为猴市），策略将直接触发警报，强制平掉所有 A 股仓位，切入 ETF 防御模式，并**挂起（Pause）后续的月度开仓与周度轮动模块**，防止在震荡市中被反复“左右打脸”。

### 2. 模块 1：月度动量研判与调仓 (每月初 09:35)
* *(仅在非猴市且未被挂起时执行)*
* 计算大盘（沪深300）与小盘（中证1000）的复合平滑动量（50%的10日涨幅 + 50%的20日涨幅）。
* **双防机制**：大盘小盘动量均为负时，清仓 A 股买入防守 ETF。
* **风格轮动**：动量为正时，比较强弱，决定当月做多“大盘白马”还是“微盘绩优”。

### 3. 模块 2：周度均线熔断 (每周五 14:30)
* *(仅在非猴市且未被挂起时执行)*
* 提取当前风格基准指数的 20 日均线 (MA20)。若周五尾盘指数跌破均线，提前清仓 A 股切入 ETF，规避周末可能发酵的暴跌风险。

### 4. 模块 3：日内硬止损 (每日 14:45)
* *(全局放行，不受猴市挂起限制)*
* 遍历策略名下所有持仓，监控最新 Tick 现价。
* 若现价跌破持仓成本的 8%，触发刚性止损，市价一键清仓该股，且止损后的资金保留为空仓（不向下摊平）。

### 5. 选股清洗逻辑 (`_filter_fundamentals`)
* **排雷过滤**：剔除 ST 股、退市股，且强制要求**扣非净利润 > 0**。
* **大盘风格 (BIG)**：ROE > 10% 且 0 < PE(TTM) < 30，按总市值从大到小排序。
* **小盘风格 (SMALL)**：ROE > 15%，按总市值从小到大排序。

## ⚖️ 策略的优缺点

### 优点
1. **具备环境感知能力**：新增的猴市识别机制（ER+CV）有效解决了传统均线/动量策略在宽幅震荡市中频繁亏损手续费和滑点的致命缺陷。
2. **多维立体风控**：震荡过滤（日）、均线防暴跌（周）、动量大择时（月）、8% 个股硬止损（日内），形成了极具韧性的资金保护网。
3. **状态持久化**：依赖 JSON 进行状态与账本管理（`StateManager`, `StrategyLedger`），支持随时断线重连或重启，无惧断电。

### 缺点
1. **阻塞式等待遗留**：调仓时依然使用了 `time.sleep(20)` 来等待资金结算，这 20 秒内策略无法响应其他 Tick 级别的数据（例如触及止损的股票）。
2. **滞后性风险**：猴市识别基于过去 20 天的滚动数据，当系统解除“猴市预警”时，市场可能已经走完了第一波主升浪的启动阶段。
3. **市价单滑点**：极端行情下，`xtconstant.LATEST_PRICE` 可能产生较大滑点。

## ⚙️ 策略的关键参数

建议根据你的资金体量和风险承受能力，在 `__init__` 与辅助函数中调整以下参数：

| 参数类别 | 参数名 | 当前设定值 | 描述说明 |
| :--- | :--- | :--- | :--- |
| **资金与标的** | `stock_num` | 3 | A 股目标持仓数量。 |
| | `total_budget` | 60000 | 策略最大使用资金（元）。 |
| | `benchmark_big` | '000300.SH' | 大盘基准及猴市判断基准（沪深300）。 |
| | `benchmark_small`| '000852.SH' | 小盘基准（中证1000）。 |
| | `foreign_etf` | ['518880', '513100'] | 避险资产：黄金 ETF 与 纳指 ETF。 |
| **时间控制** | `circuit_breaker_time` | "14:30:00" | 周五均线熔断检查时间。 |
| | `stop_loss_time` | "14:45:00" | 每日 8% 硬止损检查时间。 |
| **猴市识别** | `window` | 20 | 考夫曼 ER 指标计算的回溯周期。 |
| | `er_threshold` | 0.25 | 效率系数阈值（<0.25 视为无趋势）。 |
| | `vol_threshold` | 0.015 | 变异系数阈值（>0.015 视为高波动）。 |

## 🎯 策略适合的场景

* **极度舒适**：**单边牛市**（稳吃主升浪） 与 **单边熊市**（早早切入 ETF 避险或吃外盘红利）。
* **安全规避**：**宽幅震荡的“猴市”**。系统会识别出乱局，主动挂起开仓指令，抱紧防御 ETF 躺平，成功避免了“左右打脸”。
* **潜在盲区**：**急速的深 V 反转**。如果个股在盘中突然遭遇暴风雨式洗盘（跌幅超 8%），会在 14:45 被无情止损，即便第二天迅速反包也无法接回。

## ⏱️ 策略跑的频率

* **引擎频率**：依赖 QMT 极简模式本地客户端，死循环 `while True` 驱动，每 **3 秒**轮询一次。
* **业务频率（极低频，适合挂机）**：
  * **09:31**：每日 1 次（计算猴市环境）。
  * **09:35**：每月 1 次（动量调仓，仅在非挂起状态执行）。
  * **14:30**：每周五 1 次（破位熔断，仅在非挂起状态执行）。
  * **14:45**：每日 1 次（日内持仓硬止损）。
