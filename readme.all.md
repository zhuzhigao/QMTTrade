# 金阳光 QMT 极简模式量化策略集 (XtQuant)

本仓库包含了四套专为**光大证券金阳光 QMT（极简模式）**开发的 Python 量化交易策略。策略涵盖了从高频趋势动量到低频资产配置的不同维度，旨在构建一个完整的量化投资组合。

## 🛠 环境配置
- **交易终端**：光大证券金阳光 QMT (极速版)
- **运行模式**：极简模式 (Mini Mode)
- **Python 库**：`xtquant`, `pandas`, `numpy`, `scipy`
- **资金账号**：`47601131`
- **本地支持**：部分策略依赖本地 SQLite 数据库（用于存储财务因子）

## 📂 策略概览

### 1. kj202509：大小盘动量与基本面双驱
**【策略逻辑】**
通过监控沪深 300（大盘）与中证 1000（小盘）的相对强弱决定投资风格，并利用严苛的财务指标（ROE、扣非净利等）筛选标的。引入“猴市过滤器”在无序震荡期强制空仓避险。

* **核心机制**：
    * **风格择时**：月度比较动量，弱势时切换至纳指 ETF 或黄金。
    * **日内巡检**：`is_monkey_market` 函数动态识别震荡行情。
    * **硬性止损**：个股 8% 阈值止损，周线破位熔断。
* **适用环境**：主线清晰的单边趋势市（无论牛熊）。
* **风险提示**：在风格频繁切换的“电风扇”行情中可能出现连续磨损。

### 2. kj202536：RSRS 择时与多资产 ETF 轮动
**【策略逻辑】**
基于高阶统计学的 RSRS（阻力支撑相对强度）指标作为总开关，配合“平滑动量打分系统”，在商品、红利、宽基及海外资产中挑选走势最平稳的 3 只 ETF 进行配置。

* **核心机制**：
    * **RSRS 算法**：利用最高价/最低价回归斜率的 Z-Score 识别市场合力方向。
    * **平滑度优先**：收益率 × R²（拟合优度），优先选择“每天涨一点”而非“大起大落”的标的。
    * **类别分散**：强制跨资产类别配置，天然对冲单一行业风险。
* **适用环境**：宏观经济周期清晰、大类资产有序轮动的市场。
* **风险提示**：极端普跌或流动性枯竭环境下，由于反应时滞可能产生短期净值回撤。

### 3. kj202579：高股息基本面极致防雷
**【策略逻辑】**
深度价值投资策略。以“不亏钱”为第一准则，通过避开财报季高风险期、严查审计意见以及本地数据库筛选，构建具备极高安全边际的高分红组合。

* **核心机制**：
    * **财报季回避**：1 月、4 月强制空仓 A 股，转入货币基金，完美规避“业绩雷”。
    * **逆势仓位管理**：根据沪深 300 乖离率反向调节持股数量（超卖加仓，超买收缩）。
    * **小黑屋制度**：止损个股进入 30 天禁买期，防止情绪化反复进场。
* **适用环境**：经济复苏初期或存量博弈的“价值/红利”主导市场。
* **风险提示**：在估值中枢下移或题材疯涨的“水牛”行情中，可能长期跑输大盘。

### 4. kj202590：固收+ ETF 再平衡
**【策略逻辑】**
纯粹的资产配置策略。通过固定比例（70% 债券 + 30% 风险资产）构建防御阵地，通过纪律性的低买高卖（再平衡）赚取资产波动的收益。

* **核心机制**：
    * **静态配置**：国债 (70%) + 黄金 (14%) + 红利 (8%) + 纳指 (8%)。
    * **偏离触发**：仅当市值偏离目标权重 > 15% 时执行交易，减少无谓的手续费开支。
    * **不对称止损**：对权益部分设止损，对避险资产长线持有。
* **适用环境**：全天候环境。尤其适合作为底仓配置或在降息周期中持有。
* **风险提示**：极端罕见的“股债双杀”情形下，再平衡机制会导致短期越跌越买。

## 🚀 使用指南

1. **启动终端**：打开“光大证券金阳光QMT极速版”，登录资金账号 `47601131`。
2. **配置路径**：在代码 `Config` 类中修改 `mini_qmt_path` 为你本地的 `userdata_mini` 路径。
3. **模式切换**：
    * **调试模式**：使用 `python kj202509.py -m DEBUG`（仅日志输出，不报单）。
    * **实盘模式**：使用 `python kj202509.py -m REAL`（真实资金触发报单）。
4. **数据维护**：79 号策略运行前，请确保本地 `stock_data.db` 已完成更新。
